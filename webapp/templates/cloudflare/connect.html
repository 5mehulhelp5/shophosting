Connect Cloudflare
Manage your DNS records directly from your dashboard
How to Create Your API Token
  1. Add your domain to Cloudflare (if not already added):
    Go to Cloudflare Dashboard , click "Add a Site", enter your domain, and follow the setup wizard. You'll need to update your domain's nameservers at your registrar to point to Cloudflare.
    (Skip this step if your domain is already on Cloudflare)
  2. Go to Cloudflare API Tokens (My Profile → API Tokens).
  3. Click "Create Token" button.
  4. Find "Edit zone DNS" template and click "Use template".
    This template includes:
    Zone - DNS - Edit - Manage DNS records
    Zone - Zone - Read - View zone information
  5. Under "Zone Resources", select:
    Include → Specific zone → Your domain
    (Or select "All zones" if you want to manage multiple domains)
  6. Click "Continue to summary", then "Create Token".
  7. Copy the token immediately - it won't be shown again!
Your token will be encrypted and stored securely.

Your token is safe. We only request the minimum permissions needed to manage DNS records. Your token is encrypted using industry-standard encryption before being stored. You can revoke access anytime from your Cloudflare dashboard.
